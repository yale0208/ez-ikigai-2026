avg = (Passion + Mission + Calling + Mastery) / 4
maxPossible = 每軸理論最高值
energyRatio = avg / maxPossible

| 動能等級   | 條件                      | 狀態解讀           | 主要風險         | 建議方向          |
| ------ | ----------------------- | -------------- | ------------ | ------------- |
| 🔥 高動能 | energyRatio ≥ 0.7       | 投入度高、角色清晰、行動力強 | 若失衡易 Burnout | 強化界線與節奏管理     |
| ⚖️ 中動能 | 0.4 ≤ energyRatio < 0.7 | 穩定投入、可持續發展     | 成長速度較慢       | 可刻意強化弱軸       |
| 🌱 低動能 | energyRatio < 0.4       | 動力不足或角色錯配      | 迷惘、耗竭        | 優先恢復精力，不急調整職涯 |

gap = maxValue - minValue
gapRatio = gap / avg

| 平衡等級    | 條件                     | 狀態說明    | 風險強度 |
| ------- | ---------------------- | ------- | ---- |
| 🟢 高度平衡 | gapRatio ≤ 0.10        | 四面向整合度高 | 低    |
| 🟡 輕度失衡 | 0.10 < gapRatio ≤ 0.25 | 有重心偏向   | 中    |
| 🔴 顯著失衡 | gapRatio > 0.25        | 結構性傾斜   | 高    |

Top = 最大值
Second = 第二大
dominanceRatio = (Top - Second) / avg
